## One patient left
**Tagline:** מתעוררים לבד בבית חולים ריק — שורדים, מתגנבים, ומגלים מה קרה למשפחה.

### מהות המשחק
משחק הישרדות־סיפור 2D בעולם פוסט־אפוקליפטי אחרי מגפת “זומבים”. השחקן מתעורר בבית חולים נטוש, אוסף רמזים, נמנע מעימותים ומתקדם לשכונת מגורים סמוכה כדי להרכיב את סיפור המשפחה. דגש על התגנבות, משאבים דלים, וחידות סביבתיות קצרות.  
**פלטפורמות:** Windows (אופציונלי Web לדמו).

### רכיבים רשמיים
1. **שחקנים**  
   - קהל יעד: 13+, אוהבי נרטיב/מתח/Stealth.  
   - מצב משחק: יחיד; אינטראקציה מול מערכת (אין PvP/Coop ב־MVP).

2. **יעדים**  
   - ראשיים: לברוח מבית החולים; לאתר את דירת המשפחה; לחשוף 3 רמזי־עלילה מרכזיים.  
   - הצגה: יומן משימות קצר (Journal) + סימון נקודות עניין במפה.

3. **תהליכים**  
   - התחלה: חדר טיפול → איסוף פנס/תחבושת → דלת נעולה (הדרכת התגנבות/זחילה).  
   - ליבה: חקירה שקטה → איסוף משאבים/רמזים → עקיפה דרך אור/צליל → פתיחת נתיבים.  
   - סיום: הגעה לדירה, חידת מנעול קצרה, יציאה לאזור הבא.  
   - פקדים: WASD, **Shift** ריצה, **Ctrl** התכופפות, **E** אינטראקציה, **F** פנס.  
   - הדרכה: Tooltips קצרים + טיפי Stealth במסכי טעינה.

4. **חוקים**  
   - מד רעש/אור; קו ראייה ושמיעה לאויבים.  
   - מלאי משבצות מוגבל; ללא נשק חם ב־MVP.  
   - פציעה מורידה מהירות ומעלה רעש־בסיס.  
   - למידה הדרגתית (סצנות הדרכה + הודעות מערכת קצרות).

5. **משאבים**  
   - בריאות, תחבושות, סוללות לפנס, חפצי הסחה (בקבוק/טיימר), מפתחות/כרטיסים.  
   - השגה: ארונות/מחסנים, חידות קטנות, אזורים בסיכון.  
   - נדירות: דרופ מוגבל; טרייד־אוף בין רפואה להסחה.  
   - UI: HUD מינימלי + מסך מלאי.

6. **עימותים**  
   - מול מערכת: פטרולים, אזורי אור/צל, אזעקות.  
   - מול אחרים: אין.  
   - מול עצמי: בחירה בין סיכון (לוט) לבין חשיפה (רעש/אור).

7. **גבולות (מרחב/מפה)**  
   - פתוח־למחצה: בית חולים → סמטה שירות → שכונת מגורים.  
   - 2D שטוח; מפות מחוברות; חסימות פיזיות ברורות.  
   - עקרונות: משמעות (חדרים פונקציונליים), ניידות (עקיפות), התמצאות (שילוט/צבעים), עניין (חדרי לוט־סיפור), הכוונה (מקורות אור/צלילים).

8. **תוצאות**  
   - הצלחה: יציאה מאזור + גילוי 3 רמזים מרכזיים.  
   - כישלון: תפיסה/עילפון → טעינה מנקודת שמירה.  
   - כישרון מול מזל: ~70% שליטה/קריאה נכונה, ~30% אקראיות לוט/מסלולי פטרולים.

### סקירת משחקים קיימים
- **Darkwood** — הישרדות מתוחה; שימוש באור/צלילים. נבדל: נרטיב אישי־משפחתי מודגש, stealth קריא עם מד רעש/אור.

- **This War of Mine** — סיכון־תמורה והתגנבות; נבדל: אויבים “זומביים” ומבנה אזורי קצר.

- **Project Zomboid** — סימולציה עמוקה; נבדל: היקף קטן ומכוון־סיפור במקום סנדבוקס כבד..
